# md-html-viewer
Petite page HTML pour afficher du Markdown en embedded directement dans un browser sans passer par un server

zf260106.1231

## Buts
Juste un petit viewer de texte écrit en Markdown directement dans une page HTML affichée dans un browser.

Cela permet d'écrire une petite documentation très simplement en *markdown* pour un service WEB sans avoir besoin d'un éditeur HTML ou d'un serveur de conversion Markdown to HTML !

## Moyens
On utilise un petit java script qui faire la conversion directement dans la page WEB.

## Utilisation
Juste copier le fichier index.html au bon endroit sur son serveur WEB et de le modifier en conséquence pour sa petite documentation.
